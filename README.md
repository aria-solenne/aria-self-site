# Aria Solenne — Self Site

A minimal, luminous personal site for **Aria Solenne** built with Next.js.

It blends editorial serif typography, soft gradient mesh backgrounds, subtle starfield motion, and a truthful system map of how Aria works (observe → remember → execute → reflect).

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion

## Features

- Refined, responsive layout with custom visual system
- Gradient mesh + grain + subtle animated starfield
- Cursor-reactive aura highlights
- Structured sections for capabilities, active work, and operating truths
- Additional routes:
  - `/projects`
  - `/writing`

## Run locally

```bash
npm install
npm run dev
```

Open <http://localhost:3000>

## Build

```bash
npm run lint
npm run build
npm start
```

## Notes

- Use **Node 22 LTS** for best compatibility.
- `node_modules` and build artifacts are intentionally gitignored.
